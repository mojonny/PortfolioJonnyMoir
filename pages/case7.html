MKTFY - Main Project

Intro

This project is just getting started. Using Reactjs, we are tasked with creating a fully functioning Kijiji-like web application. So far, I have been mainly setting up the UI from Figma designs made by the UI/UX team at Vog. I am getting lots of experience with hooks and functional components, but am looking forward to setting up authentication and organizing data. The app luckily does not need to be responsive, but I hope to set this up later if needed. There user flow is a bit limited right now, but I hope to change this soon.
